A. Соревнование по списыванию (4 балла)
ограничение по времени на тест 6 s
ограничение по памяти на тест 256 MB
ввод standard input
вывод standard output

Злобный преподаватель кафедры высшей математики Подстрочкин устроил соревнование по списыванию. Из работы отличника он взял n слов, состоящих только из строчных букв латинского алфавита. Некоторые из них могут совпадать, так как работа отличника была достаточно большой. Каждый раз, когда Подстрочкин встречает i-е из известных ему слов отличника как непрерывную подстроку работы участника, он начисляет ему ai очков. Эти подстроки могут пересекаться, к примеру, в тексте aaab он встретит строку aa дважды, а строку ab один раз.

Подстрочкину не хочется читать слишком уж длинные списанные тексты, работа участника может содержать до l символов.

Ленивый студент узнал все слова отличника, которые выбрал Подстрочкин. Какое максимальное количество очков в соревновании он может набрать, если за победу ставится отл(3)?

Изначальное количество очков каждого участника равно 0.

Входные данные
Первая строка входных данных содержит два целых числа n и l (1<=n<=200,1<=l<=1014) — количество слов отличника и максимальная длина работы.
Во второй строке содержатся n целых чисел a1,a2,...,an (1<=ai<=100), означающих, что количество очков участника увеличивается на ai каждый раз, когда Подстрочкин видит i-е слово отличника.
Следующие n строк содержат слова отличника. i-ая из них содержит слово si, состоящее только из строчных букв латинского алфавита. Сумма длин слов отличника не превышает 200.
Все слова во входных данных не пустые.

Выходные данные
Выведите единственное целое число — максимально возможное количество очков в соревновании.

Примеры
Входные данные
3 6
3 2 1
heart
earth
art
Выходные данные
6

Входные данные
3 6
3 2 8
heart
earth
art
Выходные данные
16

Примечание
Оптимальный ответ в первом примере из условия — hearth, содержащий каждое слово отличника ровно по одному разу.
Оптимальным ответом во втором примере из условия является строка artart.
